# ⚡ BLACK PRO CYBER BOT

একটি সম্পূর্ণ Telegram Referral Bot।

## 📦 Install করুন

```bash
pip install -r requirements.txt
```

## ▶️ Run করুন

```bash
python bot.py
```

## ⚙️ Config

`bot.py` ফাইলের উপরে এই লাইনগুলো আপনার তথ্য দিয়ে বদলান:

```python
BOT_TOKEN = "আপনার বট টোকেন"
BOT_USERNAME = "আপনার বট username"
ADMIN_IDS = [আপনার user id]
CHANNEL_1 = "@আপনার চ্যানেল"
CHANNEL_2 = "@আপনার চ্যানেল"
```

## 📱 Admin Commands

- `/admin` — Admin Panel খুলুন
- `/cancel` — যেকোনো operation বাতিল করুন
- `/help` — Help দেখুন
